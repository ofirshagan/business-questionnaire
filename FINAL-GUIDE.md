## 🚀 הוראות הפעלה (3 צעדים בלבד):

### צעד 1: עדכן את Google Apps Script
1. פתח את [Google Apps Script](https://script.google.com) שלך
2. **מחק את כל הקוד הקיים**
3. העתק את כל התוכן מהקובץ **`google-apps-script-final.gs`**
4. הדבק ולחץ **שמור** (Ctrl+S)
5. לחץ **Deploy → New Deployment**
6. בחר **Web app** → **Anyone** → **Deploy**

### צעד 2: עדכן את הקבצים
1. **Google Analytics ID:**
   - פתח את `index.html`
   - חפש `G-XXXXXXXXXX` (שורה 33)
   - החלף ב-ID שלך מ-Google Analytics

2. **Calendly (אופציונלי):**
   - חפש `calendly.com/shaganofir` (שורה 2158)
   - החלף בלינק שלך

### צעד 3: העלה ל-GitHub Pages
1. צור repository חדש ב-GitHub
2. העלה את כל הקבצים:
```bash
git init
git add .
git commit -m "Initial deployment"
git remote add origin https://github.com/YOUR_USERNAME/business-questionnaire.git
git push -u origin main
```
3. לך ל-**Settings → Pages**
4. בחר **Source: Deploy from branch (main)**
5. המתן 2-3 דקות
6. הכתובת שלך: `https://YOUR_USERNAME.github.io/business-questionnaire/`
