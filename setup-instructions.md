# הוראות התקנה והפעלה - שאלון אפיון עסק

## סקירה כללית
יצרתי עבורך אפליקציית Progressive Web App (PWA) מלאה לשאלון אפיון העסק. האפליקציה כוללת:
- ממשק משתמש מלא ואינטראקטיבי בעברית
- 18 שאלות מחולקות ל-4 חלקים
- לוגיקה מותנית חכמה
- שמירה אוטומטית של התקדמות
- שליחה ישירה ל-Google Sheets
- עיצוב רספונסיבי למובייל וטאבלט
- אפשרות התקנה כאפליקציה במכשיר

## קבצים שנוצרו:
1. **questionnaire-app.html** - האפליקציה המלאה
2. **google-apps-script.gs** - קוד לקבלת הנתונים ב-Google Sheets
3. **manifest.json** - הגדרות PWA
4. **sw.js** - Service Worker לתמיכה אופליין

---

## שלב 1: הכנת Google Sheets

### יצירת הגיליון:
1. היכנס ל-Google Drive שלך
2. צור Google Sheet חדש
3. תן לו שם: "תשובות שאלון אפיון עסק"

### הגדרת Google Apps Script:
1. בגיליון, לחץ על **Extensions → Apps Script**
2. מחק את כל הקוד הקיים
3. העתק את כל הקוד מהקובץ **google-apps-script.gs**
4. הדבק אותו ב-Apps Script Editor
5. בשורה 154, החלף את `your-email@gmail.com` באימייל שלך
6. לחץ על **Ctrl+S** או על כפתור השמירה

### פרסום ה-Web App:
1. לחץ על **Deploy → New Deployment**
2. לחץ על הגלגל שיניים ובחר **Web app**
3. הגדר:
   - Description: "Questionnaire Form Handler"
   - Execute as: **Me**
   - Who has access: **Anyone**
4. לחץ **Deploy**
5. אשר את ההרשאות (אם מופיע)
6. **העתק את ה-URL** שהתקבל (נראה כמו: https://script.google.com/macros/s/...)

---

## שלב 2: הגדרת האפליקציה

### עדכון ה-URL:
1. פתח את הקובץ **questionnaire-app.html**
2. חפש את הטקסט: `YOUR_GOOGLE_APPS_SCRIPT_URL_HERE`
3. החלף אותו ב-URL שהעתקת מ-Google Apps Script
4. שמור את הקובץ

## שלב 3: העלאה לאחסון

### אפשרות א' - Netlify (חינם ומהיר):
1. היכנס ל-[Netlify](https://www.netlify.com/)
2. הירשם חינם
3. גרור את כל 4 הקבצים לאזור ה-Drop
4. המתן להעלאה
5. קבל כתובת URL לאפליקציה שלך!

### אפשרות ב' - GitHub Pages (חינם):
1. צור repository חדש ב-GitHub
2. העלה את כל הקבצים
3. לך ל-Settings → Pages
4. הפעל GitHub Pages
5. בחר branch: main

### אפשרות ג' - אחסון רגיל:
1. העלה את כל הקבצים לשרת שלך
2. ודא שהשרת תומך ב-HTTPS (חובה ל-PWA)

## שלב 4: בדיקת האפליקציה

### בדיקה ראשונה:
1. פתח את האפליקציה בדפדפן
2. מלא את השאלון לדוגמה
3. בדוק שהנתונים הגיעו ל-Google Sheet

### בדיקה במובייל:
1. פתח את הכתובת במובייל
2. בדוק שהעיצוב נראה טוב
3. נסה להתקין כאפליקציה (Add to Home Screen)

---

## שלב 5: ניתוח הנתונים

### ב-Google Sheets:
- כל תשובה חדשה תופיע בשורה חדשה
- אפשר לסנן, למיין ולנתח
- אפשר ליצור גרפים ודוחות

### קבלת התראות:
- כל ליד שמעוניין בייעוץ ישלח אליך מייל אוטומטי
- המייל יכלול את כל פרטי הליד

### פונקציות ניתוח מובנות:
בתוך Apps Script יש פונקציות נוספות:
- `getStatistics()` - סטטיסטיקות בסיסיות
- `generateAnalysisReport()` - דוח ניתוח מלא
- `exportLeadsToCsv()` - ייצוא ל-CSV

כדי להריץ אותן:
1. פתח את Apps Script
2. בחר פונקציה מהתפריט
3. לחץ Run
